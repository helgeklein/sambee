Sambee is designed for fast file exploration, not just one-file-at-a-time viewing.

## Explore Folders Efficiently

Use the normal file-browser controls to:

- open folders from the current list
- move back toward the root with breadcrumbs or parent-directory navigation
- switch between list and details views when you need more or less detail on screen
- sort by the field that best fits the task, such as name, size, modified time, or type

For fast directory jumping, commands, and current-pane filtering, see [Smart Navigation and the Quick Bar](../smart-navigation-and-the-quick-bar/).

## Use Dual Pane on Desktop

On desktop, Sambee supports a dual-pane layout with two directory panes side by side.

Dual pane is useful when you want to:

- compare two folders at once
- prepare a source in one pane and a destination in the other
- copy or move items more deliberately
- keep one pane on a reference folder while you work in another

The active pane matters. Commands and keyboard-driven actions target the pane that currently has focus.

## Copy and Move in Dual Pane

Dual pane is the clearest setup for copy and move work because the destination is visible at the same time as the source.

In practice, that means you can:

- open the source folder in one pane
- open the destination folder in the other pane
- confirm the destination visually before starting the operation

Some copy or move combinations may still be unavailable depending on the source and destination connection. If Sambee cannot perform the requested operation safely, it shows an error instead of silently changing the data.

## Desktop Versus Mobile

Dual pane is a desktop feature. On compact or mobile layouts, Sambee uses a simpler single-pane navigation model.

If you need desktop-style pane workflows, switch to a desktop browser.
