Sambee can preview supported files directly in the browser so you can inspect content before downloading it.

Start with:

- [Preview Supported Files](./preview-supported-files/): images, PDFs, Markdown, and the normal browser preview path
- [What Happens When A File Cannot Be Previewed](./what-happens-when-a-file-cannot-be-previewed/): how to choose the right fallback when Sambee cannot show the file in the browser

Use this section when the question is whether Sambee can show a file in-browser or which fallback path to take when it cannot.

For the stable reference summary of preview coverage, see [Supported File Formats](../reference/supported-file-formats/).
