## Browser-Based File Manager

Sambee is a browser-based viewer and manager for files on SMB network shares and on your computer's local drives. It supports single and dual-pane views and can be a replacement for Total Commander-style tools - in your browser and on your phone.

## SMB Shares and Local Drives

SMB shares are accessed via Sambee's server component, which translates between HTTP(S) (which your browser understands) and SMB (which a file server understands). Local drive access is provided by Sambee Companion, a helper app that can be installed on your computer and serves as a micro-server your browser can talk to.

All of the above means that Sambee brings your files and folders into the browser and makes them accessible from your phone in the same way as from your PC or Mac.

## Image and PDF Viewer

But Sambee is much more than that. It's file management tool built for speed, a fast image browser with broad file type support, a Markdown editor, and a PDF viewer. All of the above work directly in the browser and don't require additional software (except Companion for accessing local drives).

## Markdown Editor

But it doesn't stop there. Sambee bridges the gap between the browser and the natively installed apps on your computer: Sambee lets you open any file in its traditional desktop app directly from the browser. In other words, it lets you open Word, Excel, or Photoshop files in their respective native apps. When you're done editing, your changes are uploaded back into the original storage location (e.g., a file share). All that happens smoothly in a friendly UI.
