The exact layout of Sambee's UI can vary by screen size, permissions, and whether you are using one pane or two, but the same basic navigation model stays the same.

## 1. Connection List

This is where you choose what storage connection you want to work in.

Depending on your setup, you may see:

- SMB connections
   - shared connections configured by an administrator
   - private connections that belong only to your account
- local drives from your computer

Keyboard shortcut: <kbd>Ctrl</kbd> + <kbd>Down</kbd>

## 2. Breadcrumbs and Folder Navigation

Breadcrumbs show where you are in the current connection's directory tree.

Use them to:

- confirm your current folder
- jump back to a parent folder
- understand which pane or connection you are working in

## 3. File List

This is the main part of the user interface. It lists the files and folders in the currently selected directory.

Use it to:

- navigate into folders
- open previewable files
- select files for actions such as download, rename, move, or delete
- inspect modification times, sizes, or other details

## 4. File Actions and Preview

When you select or open a file, Sambee shows the actions that are safe for that file type and your permissions.

Common next actions include:

- preview in the browser
- download
- rename or move
- edit Markdown in the browser
- open in a desktop app through **Open in App**

If a file type is previewable, Sambee opens a viewer instead of forcing you to download first.

## 5. Search, Filtering, and Faster Navigation

Sambee includes tools for moving through folders more quickly than repeated clicking.

Depending on the page and layout, you may see:

- search or quick navigation controls
- sorting controls
- view-mode controls
- shortcuts for keyboard-driven use

Use these when you already know roughly what you are trying to reach and want to reduce manual browsing.

## 6. Dual Pane on Desktop

On desktop, Sambee can show two directory panes side by side.

This is useful when you want to:

- compare two folders
- keep a reference folder open while working elsewhere
- prepare a source and destination for copy or move work

The active pane matters. Keyboard-driven actions and many file operations apply to the pane that currently has focus.

For deeper navigation patterns, continue to [Dual-Pane Mode](../../browsing-and-navigation/dual-pane-mode/).

## 7. What Changes on Mobile

On a phone or tablet, Sambee usually uses a simpler single-pane layout.

Expect a tighter interface with less room for side-by-side work. Desktop-oriented workflows such as local drives and desktop-app editing may be unavailable or less prominent.

For the mobile-specific experience, use [Mobile Navigation](../../browsing-and-navigation/mobile-navigation/).

## A Quick Orientation Walkthrough

Use this short tour when you want to get comfortable without committing to a larger task yet:

1. Choose a connection.
2. Open one folder.
3. Use the breadcrumbs to move back up.
4. Open a supported file to see how preview works.
5. Return to the file list and inspect which actions are available for that file.

That short loop teaches the core Sambee pattern: choose a connection, move through folders, inspect a file, then take the next safe action.
