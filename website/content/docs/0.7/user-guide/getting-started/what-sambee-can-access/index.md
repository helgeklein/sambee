Start with one question: where does the file come from?

- In a network folder you reach through SMB? Sambee connects to it through its server and shows it in the browser.
- On your computer, such as Desktop, Downloads, or Documents? Install and run Sambee Companion, then pair this browser so those local drives appear in Sambee.
- Need to open the file in Word, Photoshop, LibreOffice, or another desktop app? Start from the browser with **Open in App**. Sambee Companion on the same computer handles the desktop-app handoff.
- On a phone or tablet? Stay with browser tasks such as exploring, previewing, and downloading.

## Key Terms

- **SMB share**: a network folder you reach through an SMB connection
- **Local drive**: files on your computer
- **Sambee Companion**: the desktop helper app that lets the Sambee browser app reach local files on your computer and hand files off to desktop apps

## Choose The Right Path

| If you need to... | Use this path | Notes |
|---|---|---|
| Explore a network folder through SMB | SMB share in the browser | Sambee reaches the share through its server component |
| Explore files on your computer | Local drives in the browser with Sambee Companion | Use a desktop browser on the same computer as Companion and pair that browser |
| Preview supported images, PDFs, or Markdown | Browser preview | Preview depends on file type |
| Edit Markdown quickly | Browser editing | No Companion needed |
| Open a file in Word, Photoshop, LibreOffice, or another desktop app | **Open in App** with Sambee Companion | The browser hands the file to Companion on the same computer |
| Upload desktop edits back to the original file location | Desktop editing through Sambee Companion | Finish the upload from the **Done Editing** window |
| Use Sambee on mobile | Browser with limits | Mobile is best for exploring, previewing, and downloading |

## Common First-Time Confusions

- I am on a phone or tablet and do not see local drives.

Local drives require a desktop browser, Sambee Companion, and a browser pairing.

- I can find the file, but I want to keep working in Word or another desktop app.

Find the file in Sambee first, then use **Open in App**. Sambee Companion handles the handoff to the desktop app on the same computer.

- I do not see the share or folder I expected.

That is usually a setup or permission issue, not a browser problem.
