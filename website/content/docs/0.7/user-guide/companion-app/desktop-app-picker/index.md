The app picker lets you choose which installed desktop app should open a file.

## The App Picker

When you use **Open in companion app**, Sambee hands the request to Companion. Companion then opens an app picker dialog that lists the installed apps registered for the selected file type.

The picker groups apps into these sections:

- **Default**
- **Suggested apps**
- **More options**

### Choose or Change the App

To continue:

- Select the app you want.
- Choose **Open**.

You can also use **Choose another app** to browse to a specific executable.

### Remember the Selection

If you turn on **Always use this app** for a file type, Companion saves that executable as the preferred app for that extension.

On later opens, Companion still shows the picker, but it preselects the saved app for you. It does not silently skip the picker.

If you want to replace the remembered app for an extension, choose a different app and leave **Always use this app** turned on when you open it.
