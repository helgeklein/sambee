Sambee supports two main access paths:

- SMB shares in the browser
- local drives through Sambee Companion on the same computer

Start with:

- [Connect To An SMB Share](./connect-to-an-smb-share/): browse a network share that has already been made available to you
- [Access Local Drives And Pair Your Browser](./access-local-drives-and-pair-your-browser/): unlock local-drive access on this computer through Companion

Use this section when the next question is how to reach files, not how to browse or edit them once they are already open.

If you are not sure which path applies, go back to [What Sambee Can Access](../getting-started/what-sambee-can-access/).
