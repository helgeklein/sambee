Local drive access is available once the Companion app is [installed and paired](../../companion-app/install-and-pair-the-companion-app/).

## Explore Local Drives

If Companion is running and paired, local drives appear in the **Connection List** next to SMB connections (keyboard shortcut: <kbd>Ctrl</kbd> + <kbd>Down</kbd>).

The list includes all drives that have a drive letter:

- **Disks**, e.g., `C:\`
- **Mapped network drives**, e.g., SMB connections to a file server.
- **Virtual drives**, e.g. from Google Drive, OneDrive, or DropBox.
