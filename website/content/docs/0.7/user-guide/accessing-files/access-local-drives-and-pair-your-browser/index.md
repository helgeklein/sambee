Use this page when you want the Sambee browser app to show files on your computer.

To do that, this browser must pair with Sambee Companion, the helper app running on the same computer. Pairing means you approve the browser app to talk to Sambee Companion.

## Quick Checklist

Before you pair, make sure all of these are true:

- a desktop browser on Windows, macOS, or Linux
- Sambee Companion installed on your computer
- Sambee Companion running

Need Companion first? Start with [Install And Start The Companion App](../../companion-app/install-and-start-the-companion-app/).

## Pair This Browser

1. Open the Local Drives screen in Sambee.
2. Check the current status summary.
3. If local drives are not ready yet, choose **Pair This Browser**.
4. Verify that the code shown in the browser matches the code shown in Sambee Companion.
5. Confirm the pairing in both places.
6. Wait for the success message.

After pairing succeeds, local drives on your computer appear in Sambee for this browser.

Need desktop editing too? Continue to [Open Files In Desktop Apps And Save Changes Back](../../editing-files/open-files-in-desktop-apps-and-save-changes-back/).

## Test An Existing Pairing

If the Local Drives screen shows that this browser is already paired, run **Test Current Pairing** to confirm the browser can still reach Sambee Companion correctly.

This is useful when:

- local-drive access worked before but now seems missing
- the browser was updated or reset
- you want to confirm the current computer is still the right one for this workflow

## Pair Again If Access Stopped Working

If local drives worked before but do not work now, pair this browser again.

This usually means Sambee Companion still recognizes the browser, but the browser lost its saved pairing information. Pairing again restores the link.

## Remove Access From This Browser

If you no longer want this browser to access local drives on your computer, use **Unpair This Browser**.

After that, you must pair again before local drives will work in this browser.

## If Pairing Fails

Check these first:

- Sambee Companion is actually running on your computer
- you are pairing from a desktop browser, not an iPhone or Android browser
- the verification codes match
- the browser is allowed to communicate with Sambee Companion on your computer

If the pairing test fails after a browser was already paired, pair again to restore access.
