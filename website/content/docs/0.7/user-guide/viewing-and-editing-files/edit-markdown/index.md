Markdown files are opened in Sambee's [Markdown viewer](../view-images-and-read-documents/). From there, you can seamlessly switch to a full-fledged Markdown editor.

## Entering Edit Mode

In the Markdown viewer's toolbar, click the edit icon or press <kbd>E</kbd> to switch to the Markdown editor. It opens in rich-text mode, but you can switch to source mode if you prefer to edit raw Markdown.

## Saving Changes

A small indicator next to the file name shows whether the document has unsaved changes. Click the save icon or press <kbd>Ctrl</kbd> + <kbd>S</kbd> at any time to save the current state of the document.

## Leaving Edit Mode

Click the X icon or press <kbd>Esc</kbd> to leave the Markdown editor. If there are unsaved changes, you are asked whether to save them.
